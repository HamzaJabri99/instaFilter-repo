# insta-filter

This is one of my little projects as a beginner web-developer. it's basically a website that you can import a picture via URL and apply some filters on it,
after that you can export it as a png or jpeg.

# Framework

In this project, i use Vue.js v2.x. i made a lot of changes in the store folder (vuex).

# Skills

I've learnt so much In this app from working with multiple components also as to working with canvas and applying style through css variables and modifying them with the help of ES6.
